# MyStore - E-Commerce Catalog Website

## 📌 Project Overview
MyStore is a **responsive e-commerce catalog website** built using **HTML5 & CSS**.  
It simulates a professional shopping portal with product categories, search functionality, and interactive product cards.  
This project demonstrates front-end development skills and industry-style design, making it suitable for portfolio and resume showcase.

---

## 🛠️ Technologies Used
- **HTML5** – Semantic structure and layout  
- **CSS3** – Styling, responsiveness, hover effects  
- **Flexbox & Grid** – Modern layout techniques
